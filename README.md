# 🌐 Simple Modern Landing Page

> Uma interface web elegante e minimalista desenvolvida para destacar competências em estruturação semântica e design visual focado no usuário.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir uma **Landing Page** que combine simplicidade técnica com uma estética profissional, utilizando as melhores práticas de marcação para criar um site visualmente atraente.

O layout apresenta uma barra de navegação completa e uma seção de destaque ("Hero Section") projetada para capturar a atenção do visitante e incentivar a interação através de CTAs (Call to Action).

## ✨ Destaques e Diferenciais Técnicos

* **Navegação Semântica:** Uso da tag `<nav>` e listas estruturadas para garantir uma navegação acessível e bem organizada.
* **Interface Orientada a Objetivos:** Inclusão de botões estratégicos como "Saiba mais" e "Inscreva-se", simulando um fluxo real de conversão de usuários.
* **Integração de Iconografia Profissional:** Utilização da biblioteca **Boxicons** para elevar o padrão visual da interface.
* **Arquitetura de Layout Limpa:** Divisão clara entre a navegação superior e o conteúdo central (`center`), facilitando a manutenção e futuras expansões do código.
* **Organização de Projeto:** Separação rigorosa entre a estrutura (`index.html`) e a identidade visual localizada no diretório `css/style.css`.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e organização de conteúdo.
* **CSS3:** Gerenciamento de layout, tipografia e design visual.
* **Boxicons:** Biblioteca de ícones vetoriais para suporte visual.
